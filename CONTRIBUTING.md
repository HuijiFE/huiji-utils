# Contributing to Huiji-Utils

Huiji-Utils is a mono repository managed with [Lerna](https://github.com/lerna/lerna).\
All of packages are in the directory `packages/@huiji/`.

## Getting started

### Environment

- NodeJS >= 10.0.0
- Yarn >= 1.12.0

### Installing

```bash
lerna bootstrap
```

### Testing

Run all tests for all packages.

```bash
lerna run tests
```

### Buiding

Build all packages.

```bash
lerna run build
```
